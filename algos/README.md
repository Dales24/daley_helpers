# algos

Reference implementations of common algorithms — kept readable over clever, for interview prep and
copy-paste starting points.

| File               | Algorithm       | Complexity           |
|--------------------|-----------------|----------------------|
| `binary_search.py` | Binary search   | O(log n) time        |
| `quicksort.py`     | Quicksort       | O(n log n) average   |

Each file is runnable directly (`python3 algos/quicksort.py`) and has a small demo under
`if __name__ == "__main__"`.
